# doorphone
一个基于ARM平台的局域网门禁对讲系统
 
硬件平台 ：

1，粤嵌GEC-6818开发板（两台）

2，摄像头（JPEG/YUV）

 
软件平台 ：

1，Linux操作系统
 

功能描述：

1，两台GEC-6818互联，一台充当洋楼大堂入口的访客机，一台充当业主家里的监控机

2，访客机可进行的操作：

A) 输入房间号，激活监控机的求访铃声。（视频自动开启，且无法关闭）
 
B）关闭对讲系统。（视频自动关闭）
 
3，业主机可进行的操作：

A) 点击“监控”，可单向看到并听到大堂的实时图像和声音。

B) 点击“语音对讲”，可实时对讲，再次点击则关闭此功能。

C) 点击“视频对讲”，可实时视频对讲，再次点击则关闭此功能。

D) 点击“开门”，向来客机发送一串特定控制符，对讲结束。
